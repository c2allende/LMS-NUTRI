# Plantilla LMS - Basada en Docencia 4.0

Esta es una versión limpia del proyecto Docencia 4.0, diseñada para ser utilizada como base para nuevos cursos.

## Instrucciones de uso:
1. Configura un nuevo proyecto en Firebase.
2. Actualiza `scripts/firebase-config.js` con las nuevas credenciales.
3. Personaliza el contenido en los archivos HTML.
4. Sube las reglas de Firestore desde `firestore.rules`.
